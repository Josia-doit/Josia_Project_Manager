# CHANGELOG.md

所有版本更新记录，按时间倒序排列（最新版本在前）

## [v1.1.0] - 2026-04-12

### 优化

- 项目结构精简：删除旧版冗余垃圾文件（structure_parser.py），仅保留核心运行模块，提升程序稳定性与运行速度

- 新增一键打包EXE功能：支持单文件/多文件两种打包模式，适配不同部署需求；新增打包后自动清理临时文件功能，打包完成后自动删除 build 目录、dist 目录及 .spec 配置文件，无需手动清理，保持项目目录整洁

- 文件读写优化：增强异常处理机制，避免文件创建、删除、重命名时出现崩溃，弹出明确错误提示

- 新增批量导出功能：支持将当前项目所有代码文件批量导出为JSON格式，便于项目备份、共享及二次编辑（**注意**：项目内的图标、图像文件不会导出）

- 新增运行测试功能：添加运行测试入口，支持快速验证项目运行状态，可自行扩展测试逻辑

- 快捷键优化：为批量导出、打包EXE、批量写入等新增功能配置对应快捷键，提升操作效率

### 修复

- 修复代码编辑器颜色显示错误：解决 QsciScintilla 颜色参数类型不匹配问题，彻底解决程序启动报错问题

- 修复部分文件操作异常问题，优化编辑器启动兼容性

### 文件变化

- 功能新增：批量导出功能、运行测试功能，完善打包EXE功能细节及快捷键配置
